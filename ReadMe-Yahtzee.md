Yahtzee
=================
 
Roll dice randomly and see if you get a Yahtzee!

## TDD Test Lists

"2", "2", "2", "2", "2" = "Yahtzee"
"2","2","2","2","1" = "Four of a Kind"
"2","2","2","3","1" = "Three of a Kind"
"2","2","1","1","3" = "Two Pairs"
"2","2","3","4","5" = "One Pair"
"1","2","3","4","5") = "One of a Kind"
"2","2","2","5","5") = "Full House"
